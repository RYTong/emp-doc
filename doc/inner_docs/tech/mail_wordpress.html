Wordpress安装邮件服务

  • 环境
  • 安装插件
    • 安装php_mycrypt
  • 配置

在wordpress上安装邮件服务，以便新用户注册或评论时都能收到通知邮件。

1 环境

  • WordPress 3.9
  • Php 5.2.17

2 安装插件

搜索SMTP并安装启用。

如果出现以下错误信息，则需要安装php_mycrypt。

Please enable 'php_mycrypt' in PHP. It is needed to encrypt passwords.

Fatal error: Could not activate SMTP. in /var/www/html/tech/wp-content/plugins/>smtp/smtp.php on line 41

2.1 安装php_mycrypt

以root用户运行：

yum install php_mycrypt

重启Apache：

/usr/sbin/apachectl restart

最后启用SMTP插件。

3 配置

选择设置-SMTP，填入下面的配置并保存：

  • Host: mail.rytong.com
  • Port: 25
  • Encryption: TLS
  • Username: server.trac@rytong.com
  • Password: **

通知邮件的发送人为管理员的邮箱地址，可以在用户里面修改。

保存成功后输入自己的邮箱测试邮件是否能正常发送。
