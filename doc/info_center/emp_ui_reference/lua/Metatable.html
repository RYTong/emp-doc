getStyleByName

  • Description: 获取指定的样式值.可以获取的样式参见表1.
    对于left、right、top、bottom、width、height等和布局相关的位置样式来说，我们规定该接口返回的值是实际值根据缩放比换算后的样式值。例如：
    在320480分辨率的手机上，一个样式width=100的div内有个样式width=50的label，label的left=30。那通过该接口获取left返回30，虽然未设置right样式，但依旧可以获取right为100-50-30=20。若此时通过set接口设置right=10，那通过该接口获取right=10，获取left=100-50-10=40。
    如果在640     960分辨率的手机上，虽然实际显示像素变为了2倍，但缩放后的样式依旧以320 * 480为基准，因此上述过程返回的值不变。
  • Syntax:
    control:getStyleByName(name)
  • Parameters:
    1. name(required)
      • 类型: String.
      • 说明: 样式名称 支持样式范围详见：setStyleByName
    2. control
      • 操作的控件对象,通常为通过getElements函数获取的.
  • Return:
    • 类型: String.
    • 说明: 样式值（如果涉及到像素值则返回值带有'px'）
  • Examples:
    local input = document:getElementsByName("button");
local width = input[1]:getStyleByName("width");
print(width); -- 200px

setStyleByName

  • Description: 设置指定样式的值,可以设置的样式参见表1.
    在容器类控件样式中对字体样式(fontsize)设置只支持写在报文中，不支持脚本动态设置 设置top,left,right,bottom,width,height时,value值必须带有单位1px.
  • Syntax:
    control:setStyleByName(name, value)
  • Parameters:
    1. name(required)
      • 类型: String.
      • 说明: 样式名称
    2. value(required)
      • 类型: String.
      • 说明: 样式值
  • Return: 无

getMatrix

  • Description: 获取当前控件的矩阵，以lua table形式返回矩阵各字段值.默认为单元矩阵 目前仅支持二维矩阵. 关于矩阵详细信息参见EMP Lua Library中关于 Matrix一章. 矩阵格式如下: struct CATransform3D
    {
    float m11, m12, m13;
    float m21, m22, m23;
    float m31, m32, m33;
    };
    返回的table中将以m11,m12等为字段名,返回它们分别对应的值.
  • Syntax:
    map:getMatrix()
  • Parameters: 无
  • Return:
    • 类型: Table.
    • 说明: 3*3矩阵{m11, m12, … m33}
  • See:
    setMatrix().

setMatrix

  • Description: 设置当前控件的矩阵.通过对矩阵的修改,达到图片变换的目的.

  • Syntax:

    map:setMatrix(matrix)
  • Parameters:
    1. matrix(required)
      • 类型: Table.
      • 说明: 3*3矩阵。{m11, m12, … m33}
  • Return: 无
  • See:
    getMatrix()

setInnerHtml

  • Description: 更新当前控件内容.当页面中的部分内容包含较大的数据量或需要较长的网络请求,为了提交用户体验,通常会将部分内容优先展示给用户,而将剩余内容通过局部刷新的方式显示.setInnerHtml就是一个完成这种功能的函数,它可以在页面显示后,通过更新单个控件内容的方式实现局部刷新.（只能是容器控件div, form, table, body）
  • Syntax:
    control:setInnerHtml(content)

  • Parameters:

    1. content(required)
      • 类型: String.
      • 说明: 报文内容（也可以包含样式、脚本，包含样式则在根节点加style，脚本则加script）。 报文内容可以带有样式定义也可不带，但是要求报文的父节点 必须与要更新的控件节点名称相同。 Content的内容必须以作为根节点。

  • Return: 无

  • Examples: 更新界面上已有的Div控件，则指定的更新报文必须为：
    <content>
<style>
...
</style>
<script>
...
</script>
<div>
    ...
</div>
</content>

submit

  • Description: 表单提交。与submit按钮的提交功能相同.本函数会将指定表单中需要提交的元素内容以post方式提交到服务器.该请求默认通过加密信道加密,并自动处理服务器返回数据. 如果服务器返回页面报文,则将该报文加入缓存,并显示页面内容.如果返回数据没有页面内容(即没有body),则该报文不加人缓存,但依然执行报文中的脚本.此方法只能由form控件调用，如果当前form表单中包含密码输入框,则会将密码进行一次一密加密.
  • Syntax:
    control:submit()
  • Parameters: 无
  • Return: 无

appendChild

  • Description:
    将指定的控件节点添加到当前控件集合的最后一个子节点后（此方法仅对可包含子节点的控件起作用）。 tr不支持此操作.table中要求每一行的td个数相同,因此不能对单个tr增加子节点.
    注：v5.3及上以版本支持
  • Syntax:
    control:appendChild(control2)
  • Parameters:
    1. control2(required)
      • 类型: Userdata.
      • 类型: 待插入的子节点。
  • Return:
    Boolean，插入成功返回真，否则返回假。
  • Examples:
    local a = document:createElement("a");
local div = document:getElementsByName("div1");
div[1]:appendChild(a);

insertBefore

  • Description:
    将control1加到control的子节点control2前，如果没有control2参数，那么默认加到最后一个子节点后（此方法仅对可包含子节点的控件起作用）。 td不支持此操作.table中要求每一行的td个数相同,因此不能删除td.
    注：v5.3及上以版本支持
  • Syntax:
    control:insertBefore(control1, control2)
  • Parameters:
    1. control1(required)
      • 类型: Userdata.
      • 类型: 待插入的子节点。
    2. control2(optional)
      • 类型: Userdata.
      • 类型: 在control中的子节点。
  • Return:
    Boolean，插入成功返回真，否则返回假。
  • Examples:
    local a = document:createElement("a");
local div = document:getElementsByName("div1");
local b = div[1]:getChildren();
div[1]:insertBefore(a, b[1]);

removeChild

  • Description:
    删除当前控件集全中指定的子节点（当一个节点被删除时，其所有子节点也会被删除。）。 tr不支持此操作.table中要求每一行的td个数相同,因此不能对单个tr删除子节点.
    注：v5.3及上以版本支持
  • Syntax:
    control:removeChild(control2)
  • Parameters:
    1. control2(required)
      • 类型: Userdata.
      • 类型: 待删除的子节点。
  • Return:
    Boolean，删除成功返回真，否则返回假。
  • Examples:
    local div = document:getElementsByName("div1");
local b = div[1]:getChildren();
div[1]:removeChild(b[1]);

  • Syntax:
    control:showLoading()
  • Parameters: 无
  • Return: 无
  • See: stopLoading()

  • Return: 无

getAllAnnotations

  • Description: 返回当前地图上所有坐标点信息。（仅地图控件可调用）

  • Return:

  • Return: 无

setOnClickListener()

  • Description:
    为控件设置监听方法,监听控件被点击的行为，当控件被点击后会执行回调方法。 同一个控件可以设置多个监听方法. 注意：监听的回调会先于属性设置的onclick方法执行。
  • Syntax: 
    setOnClickListener(callback)
  • Parameters:
    1. callback (required)_
      • 类型: Function.
      • 说明: 监听函数，在控件点击后调用。该监听函数有3个参数，分别为：sender(控件本身)，name（控件的name值），value（控件的value值）。
  • Return:
    无

  • Examples:

      function btnClick(btn, name, value)       
  end

  local btn = Document:getEventElementsByName("cal");
  btn[0]:setOnClickListener(btnClick);

setOnFocusListener()

  • Description:
    为控件设置监听方法,监听控件获得焦点的行为，当控件获得焦点时会执行回调方法。同一个控件可以设置多个监听方法.
    注意：此方法仅在输入框控件有效，监听的回调会先于属性设置的onfocus方法执行。
  • Syntax: 
    setOnFocusListener(callback)
  • Parameters:
    1. callback (required)_
      • 类型: Function.
      • 说明: 监听函数，在控件点击后调用。该监听函数有3个参数，分别为：sender(控件本身)，name（控件的name值），value（控件的value值）。
  • Return:
    无

  • Examples:

      function tfFocus(btn, name, value)       
  end

  local pw = Document:getEventElementsByName("password");
  pw[0]:setOnFocusListener(tfCocus);
