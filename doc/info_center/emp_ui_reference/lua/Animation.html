使用Lua脚本实现动画按照接口定义，分为三种：Timer，Transition，Animation。

Timer接口详见：Timer，下面详细介绍Transition与Animation这两个接口：

Transition

from()

  • Syntax
    transition:from(object,transition-property).

  • parameter
    object ：需操作的目标对象；
    transition-property(table) ：起始状态属性集合.

  • discretion
    如果与to()一起使用，必须保证object为同一个对象。

to()

  • Syntax
    transition:to(object, transition-property, duration)

  • parameter
    object ：需操作的目标对象.
    transition-property(table) ：结束状态集合. duration(秒) ：动画持续时间.

  • discretion
    动画开始，达到目标transition后停止。如果与from()一起使用，必须保证object为同一个对象。

setCurve()

  • Syntax
    transition:setCurve(object,curve)

  • parameter
    object ：需操作的目标对象； curve ：动画展示速度效果.

  • discretion
    在动画开始前设置.

setRepeatCount()

  • Syntax
    transition:setRepeatCount(object,repeatCount)

  • parameter
    object ：需操作的目标对象； repeatCount ：重复次数.

  • discretion
    -1表示无限循环

translate()

  • Syntax
    transition:translate(object, x,y,duration)

  • parameter
    object ：控件对象；
    x：x轴偏移量；
    y：y轴偏移量；
    duration(秒)：动画持续时间

  • discretion
    将object控件沿x轴及y轴位移

translateX()

  • Syntax
    transition:translateX(object,x,duration)

  • parameter
    object ：控件对象；
    x：x轴偏移量；
    duration(秒)：动画持续时间

  • discretion
    将object控件沿x轴位移

translateY()

  • Syntax
    transition:translateY(object,y,duration)

  • parameter
    object ：控件对象；
    y：x轴偏移量；
    duration(秒)：动画持续时间

  • discretion
    将object控件沿y轴位移

scale()

  • Syntax
    transition:scale(object,x,y,duration)

  • parameter
    object ：控件对象；
    x：x轴方向放大倍数（宽度）
    y：y轴方向放大倍数（高度）
    duration(秒)：动画持续时间

  • discretion
    将object控件进行缩放

scaleX()

  • Syntax
    transition:scaleX(object,x,duration)

  • parameter
    object ：控件对象；
    x：x轴方向放大倍数（宽度）
    duration(秒)：动画持续时间

  • discretion
    相当于scale(x, 1)

scaleY()

  • Syntax
    transition:scaleY(object,y,duration)

  • parameter
    object ：控件对象；
    y：y轴方向放大倍数（高度）
    duration(秒)：动画持续时间

  • discretion
    相当于scale(1, y)

alpha()

  • Syntax
    transition:alpha(object,alpha,duration)

  • parameter
    object ：控件对象；
    alpha：透明度（0~1）
    duration(秒)：动画持续时间

  • discretion
    更改控件的透明度

rotate()

  • Syntax
    transition:rotate(object,angle,axis,duration)

  • parameter
    object ：控件对象；
    angle：旋转角度值
    axis：指定旋转的轴（x,y,z）
    duration(秒)：动画持续时间

  • discretion
    在2D平面上更改控件的旋转角度。（顺时针旋转）；
    axis有三个值可选：'x'，'y'，'z'

skew()

  • Syntax
    transition:skew(object,xangle, yangle,duration)

  • parameter
    object ：控件对象；
    xangle ：x轴方向斜拉角度值
    yangle ：y轴方向斜拉角度值
    duration(秒) ：动画持续时间

  • discretion
    在2D平面上更改控件的斜拉角度

skewX()

  • Syntax
    transition:skewX(object,angle,duration)

  • parameter
    object ：控件对象；
    angle ：x轴方向斜拉角度值
    duration(秒) ：动画持续时间

  • discretion
    在2D平面上更改控件x轴方向的斜拉角度。

skewY()

  • Syntax
    transition:skewY(object,angle,duration)

  • parameter
    object ：控件对象；
    angle ：y轴方向斜拉角度值
    duration(秒) ：动画持续时间

  • discretion
    在2D平面上更改控件y轴方向的斜拉角度。

matrix()

  • Syntax
    transition:matrix(object,matrixTable,duration)

  • parameter
    object ：控件对象；
    matrixTable ：table类型，最多有九个元素组成：{m11=0, m12=……, m33=0}
    duration(秒) ：动画持续时间

  • discretion
    改变空间的矩阵
    M11: x轴缩放 (默认为1)
    M22: y轴缩放 (默认为1)
    M12: y轴错切 (默认为0)
    M21: x轴错切 (默认为0)
    M31: x轴平移 (默认为0)
    M32: y轴平移 (默认为0)
    M13: x轴投影 (目前未用，默认为0)
    M23: y轴投影 (目前未用，默认为0)
    M33: 投影参数 (目前未用默认为1)

setStartListener()

  • Syntax
    transition:setStartListener(object,function)

  • parameter
    object ：控件对象；
    function：监听方法（function），回调时带object本身。如：function callback(sender) end;

  • discretion
    在动画开始前调用该监听方法

setStopListener()

  • Syntax
    transition:setStopListener(object,function)

  • parameter
    object ：控件对象；
    function：监听方法（function），回调时带object本身。如：function callback(sender) end;

  • discretion
    在动画结束后调用该监听方法

可用的能够以动画形式改变的控件属性(transition-property)集合:

Property Value Info
x float类型 相对父控件的x轴的坐标
y float类型 相对父控件的y轴的坐标
width float类型 控件自身的宽度
height float类型 控件自身的高度
matrix table类型 最多有九个元素组成：{m11=0, m12=……, m33=0}
alpha float类型 控件整体的透明度
  • 注：集合中的matrix与其它属性“x,y,width,height”目前不支持组合使用，若同时出现，则选用matrix进行操作。

Animation

newFrames()

  • Syntax
    animation:newFrames(fameTable)

  • parameter
    fameTable : 每一帧帧图片名称，播放时将按照提供的图片顺序播放。

  • discretion
    初始化一个新的动画，由play()方法启动该动画。返回新创建的动画对象。

setFrame()

  • Syntax
    animation:setFrame(object,frame)

  • parameter
    object ：动画对象
    frame ：为table类型，{x,y,width,height}

  • discretion
    frame表示动画显示区域，x、y为动画相对于父控件的显示位置（父控件由add方法设置）

setDuration()

  • Syntax
    animation:setDuration(object,duration)

  • parameter
    object ：动画对象
    duration(秒)：动画持续时间

  • discretion
    代表每一帧帧图片的持续时间

setRepeatCount()

  • Syntax
    animation:setRepeatCount(object,repeatCount)

  • parameter
    object ：动画对象
    repeatCount ：重复播放次数。如果小于等于1则默认只播放一次

  • discretion
    代表播放的次数

add()

  • Syntax
    animation:add(object,superView)

  • parameter
    object：动画对象
    superView：负责展示动画的控件

  • discretion

play()

  • Syntax
    animation:play(object)

  • parameter
    object ：动画对象

  • discretion

stop()

  • Syntax
    animation:stop(object)

  • parameter
    object ：动画对象

  • discretion

setStartListener()

  • Syntax
    animation:setStartListener(object,function)

  • parameter
    object ：动画对象
    function ：监听方法，回调时带object本身。
    如：function callback(sender)end;

  • discretion 在动画开始前调用该监听方法（必须在play方法前设置）

setStopListener()

  • Syntax
    animation:setStopListener(object,function)

  • parameter
    object ：动画对象
    function ：监听方法，回调时带object本身。
    如：function callback(sender)end;

  • discretion 在动画结束后调用该监听方法（必须在play方法前设置）

Curve

动画展示速率(curve)控制集合：

PropertyValueInfo
curvecurve.linear线性
curve.ease_in先慢后快
curve.ease_out先快后慢
curve.ease_inout先慢再快再慢

TransitionType

页面切换动画种类集合：

PropertyValueInfo
transitionTypetransitionType.flipFromLeft翻转效果，从左侧翻转（目前仅支持iOS）
transitionType.flipFromRight翻转效果，从右侧翻转（目前仅支持iOS）
transitionType.curlUp页面卷曲效果，向上卷曲（目前仅支持iOS）
transitionType.curlDown页面卷曲效果，向下卷曲（目前仅支持iOS）
transitionType.slideFromLeft页面从左向右平推，旧页面被退出屏幕，新页面从左侧被推进屏幕。
transitionType.slideFromRight页面从右向左平推，旧页面被退出屏幕，新页面从右侧被推进屏幕。
transitionType.slideFromUp页面从上向下平推，旧页面被退出屏幕，新页面从上方被推进屏幕。
transitionType.slideFromDown页面从下向上平推，旧页面被退出屏幕，新页面从下方被推进屏幕。

动画实现

下面按照功能及实现方法来介绍动画实现机制：

Timer实现动画

在Lua脚本中建立一个Timer，循环播放动画的每一帧图片。

例：

local index = 1;
local timer;
function TimerRun()
  local img = ryt:get{name="cartoon"};
  if img and #img > 0 then
    if index == 1 then
      img[1]:css("background-image", "cartoon\_01.png");
    elseif index == 2 then
      img[1]:css("background-image", "cartoon\_02.png");
    elseif index == 3 then
      img[1]:css("background-image", "cartoon\_03.png");
    elseif index == 4 then
      img[1]:css("background-image", "cartoon\_02.png");
    end
    index = index + 1;
    if index > 4 then
      index = 1;
    end
  end
end
timer = ryt:setInterval(0.4, true, TimerRun);

简单动画

这里简单动画的概念是根据提供动画的每一帧图片，设置显示间隔时间进行展示。详见：Animation: newFrames()

属性更新动画

设置控件的属性，可设置起始属性及结束属性，或只设置结束属性，以动画的形式达到所要求属性的效果；或者通过特定的方法直接达到目标属性。 主要通过Transition接口实现。

可灵活控制的动画

通过Animation接口实现动画，可以灵活控制动画的过程，开始，结束。
