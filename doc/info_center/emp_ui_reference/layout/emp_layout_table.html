1 table 报文格式

table 控件由table,tr,td标签组成. table不支持单元格合并功能,每一行td的数量要相等.并且table与tr,tr与td之间不能插入其他标签. 

<table>
<tr>
    <td>
    ...
    </td>
</tr>
<tr>
    <td>
    ...
    </td>
</tr>
<tr>
    <td>
    ...
    </td>
</tr>
</table>

2 table 布局算法

2.1 table宽度

  • 样式中指定宽度的table,以样式宽度为准.
  • 样式中没有制定宽度,table宽度根据子控件计算.table最大宽度不能超过父控件宽度减去左右边距.

2.2 tr宽度

  • tr宽度与table宽度相同.

2.3 td宽度

  • td宽度根据子控件计算得出.
  • 同一行td之间没有间距
  • 当同一行td宽度之和小于table宽度时,最后td要填充满table的宽度.
  • 同一列td宽度相同.
  • 如果table宽度不足以显示所有的td,超出部分将不会显示.

  • 同一列td,如果给定样式宽度,则以此样式宽度作为整一列的宽度.当同一列有多个样式宽度时,以样式中指定的最大宽度作为整列宽度.

    注: 当table样式中指定宽度,同时最后一列的td样式也指定宽度,此时会有三种情况:

  • td宽度之和小于table宽度,此时最后一列td样式宽度不生效,最后一列td填满table.

  • td宽度之和大于table宽度,此时最后一列td将部分显示,超出部分将被截取.
  • td宽度之和等于table宽度,正常情况.

2.4 tr 高度

  • tr样式,td样式,tr中最大高度的td,三者取最大作为tr的高度

2.5 td高度

  • td高度与tr相同

3 布局算法

  //假设参数中的size为父控件宽度-左右边距
- (CGSize)layoutSubViewsBySize:(CGSize)size{
    //应用样式宽度, table宽度以样式宽度以准.
    //如果table没有样式宽度,则以参数中的size宽度作为计算换行的依据.

    NSArray *displayCells_ //数据 记录显示的tr

    for (遍历子控件tr){
     [trControl layoutSubViewsBySize:size]//tr做布局操作
     if (!trControl.hidden){
      [displayCells_ add:trControl]; //将显示的tr加入displayCells_
      }
    }

        //单元格对齐
    float posX = 0.0;//i列的起始坐标
    float maxWidthForCol = 0.0;//列最大宽度
    float totalWidthForCols = 0.0;//总的宽度
    float totalHeightForRows = 0.0;//总的高度
    maxTdCells // 列数

    //计算单元格宽度
    for (遍历每一列maxTdCells){
         maxWidthForCol = 获取当前列最大宽度;
         当前列td宽度 = maxWidthForCol;
         当前列td的X坐标 = posX;

         posX += maxWidthForCol;//累计列坐标
         totalWidthForCols += maxWidthForCol;//累计总宽度
    }

    //计算行高
    for (遍历子控件tr){
      rowHeight = 根据td计算每一行的最大高度
      if(tr有样式高度;){
         rowHeight = tr样式高 > rowHeight ? tr样式高 : rowHeight
      }
      rowHeight作为行高.将当前行及其子td的高度设置为rowHeight
      td中重新布局
      totalHeightForRows += rowHeight;//累计总高度
    }

    //计算到这里可以得到totalWidthForCols,totalHeightForRows.分别是计算后的table宽度和高度.然后根据样式中是否指定宽高,决定table的最终宽高.

    // 做table的其他的样式的处理. 比如border,separator,scrollEnabled.


}
