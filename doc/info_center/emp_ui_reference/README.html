EMP界面开发规范

本章将介绍客户端XHTML规范及LUA脚本定义

  • 基本语法
  • 报文格式

基本语法

  1. 所有的页面报文的语法和格式必须符合XHTML 1.0标准，不能使用HTML的专有语法，例如没有关闭的标签以及下述问题。
  2. 所有使用到的URL中，特殊字符（例如&，<，>）必须escape。
    例如：
    • 非法：<a href="foo.cgi?chapter=1&section=2&copy=3">...</a>
    • 合法：<a href="foo.cgi?chapter=1&amp;section=2&amp;copy=3">...</
  3. 嵌入的stylesheet和脚本需要使用CDATA(见XHTML 1.0规范：http://www.w3.org/TR/xhtml1/#4.8)。 例如： <script type="text/javascript"> <![CDATA[ ... unescaped script content ... ]]> </script>
  4. 所有的标签和属性名称必须为英文小写。不能使用大写或大小写混用。
  5. Lua脚本必须符合Lua语法。
  6. HTML的元素的所有的属性值必须加引号。比如<table border=＂1＂>。注意引号之间不能嵌套，如果需要嵌套的话用单引号和双引号区分。比如：<label value=＂rongyitong ‘内嵌带引号的内容’＂>。如果内部用双引号，外面就需要用单引号。

报文格式

<?xml version="1.0" encoding="UTF-8" ?>
<content>
    <head>
        <style>
            样式表样式定义
        </style>
        <script type="text/x-lua" src="RYTL.lua"></script>
        <script type="text/x-lua">
            <![CDATA[
                脚本定义 
            ]]>
        </script>
    </head>
    <body>
        页面报文
    </body>
</content>
