功能说明

select表示多选框，子节点option必须与select同时使用。

例: 

    <select>  
          <option>option111</option>
          <option>option222</option>
          ...
     </select>

属性

全局属性以及具体说明参见 * 属性介绍

select

enable:指定select是否可以弹出选项列表
@value string true/false
@default true

border:指定select是否有边框
@value string 0/1
@default 0

option

text:text表示option控件显示出来的值
@value string 任意字符串

注：option的文字单行显示

value:value值用来上传服务器
@value string 任意字符串

selected:设置select控件的默认选项，如果没有该属性，则默认选中第一个option
@value string selected

注：option的属性目前不支持lua中动态设置。

样式

select

全局样式以及具体说明参见 * 样式介绍

background-color
设置背景色
@format #RGB(如#FFFFFF)
@default transparent(透明) background-image
设置背景图
@format1 url(image.png)
@format2 url(local:image.png)
@format3 url(http:// | https// | ewplocal://)
_filter
设置渐变背景色或者透明度.
@format1 progid(startColorStr='#FFFFFF',endColorStr='#000000',gradientType='0',alpha='0.5')
@format2 progid(alpha='0.5')

border
设置控件边框的样式（宽度，线型，颜色）
@format1 25px solid #fff000
@format2 只写以上三项的任意一项或两项
@default 1px solid #d4d5d9

border-radius
设置控件四个边的圆角属性
@format1 25px 10px 0px 10px
@format2 10px
@default 0px(即直角)

注意：以下字体样式只对选中项起作用

注意：以下字体样式只对选中项起作用

color
设置字体颜色
@format #RGB(如#FF00FF)
@default #000000

font-size
设置字体大小
@format 10px
@default 各平台有差异

font-weight
设置字体类型
@value normal/bold
@default normal

font-style
设置字体样式
@value normal/italic
@default normal

text-align
设置文字对齐方式
@value left/center/right
@default left

option

option的样式支持区别于其他控件. 同一组的option只需要设置一个样式,并将此样式作用到同组的其他option上. 取前置标签的样式作为整组option的样式. 如下报文,则会以option2的样式作为整组option的样式.

<select>  
     <option >option111</option>
     <option class="option2">option222</option>
     <option class="option3">option222</option>
</select>

color
设置字体颜色
@format #RGB(如#FF00FF)
@default #000000

font-size
设置字体大小
@format 10px
@default 各平台有差异

font-weight
设置字体类型
@value normal/bold
@default normal

font-style
设置字体样式
@value normal/italic
@default normal

text-align
设置文字对齐方式
@value left/center/right
@default left

事件

option

onclick
option可以设置点击事件.
说明：option的点击事件只有在用户切换option控件时才触发。如下图示例,若select当前选中项为option1，用户在弹出的option选项列表中点击了option1不能触发点击事件，点击option2才能触发点击事件。
