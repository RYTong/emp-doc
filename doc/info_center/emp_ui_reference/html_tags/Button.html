按钮包括普通按钮和提交按钮两种.

功能说明

submit

submit按钮负责form表单的提交,因此它需要包含在form中.
例: <input type='submit' value='提交'></input>

关于form表单的提交参阅form标签介绍. form标签;

button

button为普通的按钮.
例: <input type='button' value='按钮'></input>

注:button默认居中,安卓平台button没有默认大小,在不写样式的情况下,根据内容计算实际大小.

属性

type:指定按钮的类型. @value string button/submit

value:指定按钮的值,同时也是按钮的title.

border:按钮是否有边框.默认为0,同时支持设置border样式,width,style,color.

enable:指定按钮是否可以点击.各个平台的不能点击效果可能会略有不同.

delay 指定按钮点击的延时时间

rightImg 指定控件右侧小图标

leftImg 指定控件左侧小图标

样式

color
按钮上文本颜色,即按钮上显示的文字颜色,支持色值格式#RGB(#FF00FF);

background-color
指定按钮背景色.如果按钮有边框则背景色不应该超出边框的显示区域.

background-image
指定按钮背景图.如果按钮有边框则背景图不应超出边框的显示区域.

font-size
设置字体大小

font-weight
设置字体类型

filter
设置渐变背景色

border-radius
设置控件四个边的圆角属性。
格式:border-radius: 25px 10px 0px 10px
四个圆角属性的值得顺序为：top_left, top-right, bottom-right, bottom-left.
如果要四个边的圆角值相同，可以用一个值表示.
如果要让四个角圆角弧度不一样，需要按顺序写出四个值。

visibility
规定元素是否可见。即使不可见的元素也会占据页面上的空间。
@value visible/hidden 默认为visible.
当与display属性同时出现，visibility覆盖display属性，display不起作用.

font-style
指定文本的字体样式
@value normal(默认)/italic(斜体)

display
指定控件的可视状态.
@value none/block.默认值为block.

position

指定控件的显示状态. @value fixed/static/toplevel.默认值为static.

布局规则

leftImg和rightImg图片的显示及大小的计算规则如下.

加载本地图片

由于本地图片可以直接获取到图片的宽高,因此计算方法相对简单.

  1. 如果图片的高度小于button样式的高度,则图片可以按照实际大小显示.
  2. 如果图片高度大于button样式高度,则图片需要压缩图片高度.使其完全显示.
  3. 如果button没有样式的限制,则图片已实际大小显示. 并与title内容共同计算button的大小.

网络图片

网络图片的属性值采用与src中相同的方式.由于图片下载为异步下载,在布局时需要先确定图片的大小.

  1. 我们建议在url中携带w,h参数. w,h作为img的宽高.
  2. 如果没有w,h参数.则根据button的高度计算图片的显示区域,以button高度绘制矩形区域.
  3. 关于button高度的确定,可以通过样式高度或者文字计算获得高度或者默认高度.

效果图

图中绿色区域为button,红色区域为button中文字区域.
文字内容应该在添加图标后的剩余区域中剧中显示.
文字与图标应该垂直剧中并在同一水平线上.

伪类

支持:active,:inactive样式. 伪类样式支持如下: color
按钮上文本颜色,即按钮上显示的文字颜色,支持色值格式#RGB(#FF00FF);

background-color
指定按钮背景色.如果按钮有边框则背景色不应该超出边框的显示区域.

background-image
指定按钮背景图.如果按钮有边框则背景图不应超出边框的显示区域.

font-size
设置字体大小

事件

onclick
按钮的点击事件.对于submit按钮,如果指定了onclick事件,则不执行submit功能.
