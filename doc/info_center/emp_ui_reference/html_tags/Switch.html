功能说明

<input type="switch" name="switch1">yes</input>
<input type="switch" name="switch1" checked="checked">no</input>

switch控件可以通过Lua设置checked属性来动态的改变其选中状态。

注:switch标签必须有name属性,且同一组控件需要有相同的name值.

注意：定义模板时前面定义的input标签永远表示“开”，后面的input标签永远表示“关”，这两个标签中的value属性值与“开”“关”没有直接关系。

使用建议

switch是一个选择控件，代表了'开','关'两个状态，一般在控制某个事物的状态的情况下使用，亦可在表示两种对立功能的情况下使用。

属性

全局属性以及具体说明参见 * 属性介绍

type
规定标签元素的类型.
@value string switch.

value
指定控件的值.
@value string 任意的字符串.

name
同一组控件需要设置相同的name.

hide
指定控件的可见状态,与样式中的display:none有相同的作用.同一个switch控件,设置其中一个标签的hide即可应用到整个控件.
@value string true/false 默认为false.

enable
指定控件是否可用,即是否响应用户操作.同一个switch控件,设置其中一个标签的enable即可应用到整个控件.
@value string true/false 默认为true.

checked
指定控件选项是否默认选中.当不写checked属性的时候,switch控件默认关闭.
默认 @value string 'checked'字符串.

onTintColor 指定控件打开状态的颜色.
@format #RGB(如#000000)
@default iOS平台为绿色，android平台为蓝色主题的渐变色

offTintColor 指定控件关闭状态的颜色.
@format #RRGGBB(如#000000)
@default iOS平台为白色，android平台为白色主题的渐变色

thumbTintColor 指定控件滑块的颜色.
@format #RRGGBB(如#000000)
@default iOS平台为白色，android平台为白色主题的渐变色

样式

全局样式以及具体说明参见 * 样式介绍

注意
同一个switch控件,其中一个标签应用class样式即可作用于整个switch控件

width
改变控件的宽度(iOS平台不支持).

height
改变控件的高度(iOS平台不支持).

display
指定控件的可视状态.
@value none/block.默认值为block.

position

指定控件的显示状态. @value fixed/static.默认值为static.

事件

onclick

控件的点击事件.注意switch执行的是改变状态之后的状态对应的onclick.例如,switch原本为关闭状态,点击后,变为开启状态,同时执行的是开启状态对应的onclick. onclick实例：

实例1:
<input type="switch" name="switch1" onclick="onclick()">yes</input>
<input type="switch" name="switch1" onclick="onclick()">no</input>

function onclick()
     local switch = document:getElementsByName("switch1");
     local checked = switch[1]:getPropertyByName("checked");
        if checked == "true" then
            window:alert("switch开启");
        else
            window:alert("switch关闭");
        end;
end;
//checked取自switch[1]，即检验的是开启状态的checked，当switch为开启状态checked == "true";switch为关闭状态checked == "false". 注意：true和false为字符串形式。

实例2:
<input type="switch" name="switch1" onclick="onclickone()">yes</input>
<input type="switch" name="switch1" onclick="onclicktwo()">no</input>

function onclickone()
    window:alert("switch开启");
end;   
function onclicktwo()
    window:alert("switch关闭");
end;
